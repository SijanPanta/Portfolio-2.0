# Portfolio‑2.0

A modern, responsive personal portfolio website built to showcase my skills, projects, experience, and academic background. Designed using **Next.js**, **TypeScript**, and **Tailwind CSS** with clean UI components and smooth navigation across sections.

This repository powers my live site deployed at [sijanpanta.com.np](https://sijanpanta.com.np), and automatically syncs updates from my deployment pipeline. It reflects my ongoing development journey and highlights my contributions in software projects, leadership roles, and community involvement.

## Features

- Built with **Next.js** for fast performance and SEO optimization
- Styled with **Tailwind CSS** for utility‑first responsiveness
- Modular component structure for easy customization
- Portfolio sections for About, Projects, Skills, Experience, and Contact
- Clean, maintainable TypeScript codebase

## Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **React Components & Utilities**

## Deployment

The site is automatically deployed via **Vercel**, keeping this repository in sync with live updates.

## Usage

Clone the repo, install dependencies, and start the development server:

```bash
git clone https://github.com/SijanPanta/Portfolio-2.0.git
cd Portfolio-2.0
npm install
npm run dev
